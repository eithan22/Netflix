# Netflix UI Clone 🎬

A **pixel-faithful Netflix landing page clone** built with pure **HTML5 and CSS3**. Replicates Netflix's visual design including the hero section, typography, color scheme, and layout — without any JavaScript or external libraries.

---

## 📋 Description

This project is a static HTML/CSS recreation of Netflix's marketing landing page. It focuses entirely on mastering CSS layout techniques (Flexbox/Grid), color matching, typography, responsive design, and attention to UI detail. No JavaScript was used — all visual effects are achieved purely through CSS.

---

## ✨ Features

- 🎨 **Netflix Visual Design** — Matches Netflix's dark theme, red accents, and typography
- 🖼️ **Hero Section** — Full-width background with centered text and CTA
- 📐 **CSS Layout** — Flexbox-based navigation and content sections
- 🔤 **Typography** — Font styles matching the Netflix design system
- 🌐 **Static HTML** — No dependencies, runs in any browser instantly
